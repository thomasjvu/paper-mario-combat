# Paper Mario: Power Bounce Action Command

A browser-based game demo replicating the Power Bounce action command mechanic from Paper Mario.

## How to Play

1. Open `index.html` in a web browser
2. Press **A** or **SPACE** to start
3. Press **A** or **SPACE** again to attack the Goomba
4. Time your press when the shrinking ring aligns with the target zone

## Controls

- **A** or **SPACE** - Attack / Action command input

## Timing Windows

- **NICE** (yellow-green): Perfect timing - 0.80-0.95 zone
- **GOOD** (blue): Good timing - 0.70-0.80 zone  
- **MISS** (red): Missed timing

The timing window gets smaller with each consecutive hit!
